Ce este pseudocodul?

Pseudocod este un limbaj prin care sunt descriși pașii dintr-un algoritm. Limbajul pseudocod conține structurile specifice unui limbaj de programare obișnuit (precum Pascal, C/C++, Basic, etc) dar este destinat a fi citit de către oameni, nu de către calculatoare.


Conceptele care intervin in algoritmii pseudocod sunt similare cu cele din limbajele de programare:

  • instrucțiuni
  • variabile
  • constante
  • operații
  • expresii

    • Dezavantajele pseudocodului sunt:

  • Lipsa digitalizarii.
  • Lipsa experimentarii.
  • Lipsa posibilitatii de testare.

    • Informatii despre Pseudo

    Pseudo este scris in limbajul de programare Java, fara a folosi librarii externe.

    Pseudo este un interpretator pentru pseudocod care este interpretat de doua ori.

    Softul Pseudo executa pas cu pas instructiunile de pseudocod aflate intr-un fisier.

    De-a lungul timpului s-au format variante ale pseudocodului, care difera prin cuvinte cheie sau reguli de sintaxa.


    Exemplu a variatiei pseudocodului:

  • si, and, &&.
  • not, !.
  • sau, or, ||.

    • Cum se instaleaza?

    Softul in sine poate fi descarcat de pe urmatorul link pentru Windows sau de pe linkul pentru MacOS.

    Aplicatia este distribuita sub forma de arhiva, care trebuie sa fie extrasa de catre utilizator intr-o locatie aleasa de catre aceasta.

    Pentru finalizarea procesului de instalare, utilizatorul trebuie sa execute cu permisiuni de administrator script-ul "setuo.bat".

    Utilizare soft

    Pseudo este un soft utilizat prin intermediul terminalului.

    Comanda pentru a interpreta codul scris in pseudocod dintr-un fisier este "pseudo <fisier>".

    Dupa ce utilizatorul a executat comanda, programul va interpreta codul sursa.
